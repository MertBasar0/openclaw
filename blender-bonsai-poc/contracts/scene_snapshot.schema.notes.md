# Scene Snapshot v0 notes

İlk kontrat intentionally small:

- source: tool/runtime metadata
- scene: object/material/collection counts + bounds
- objects: name, type, collection, dimensions, location, material names, custom properties
- materials: name + base color
- diagnostics: missing material, tiny/large object, no semantic category etc.

BIM fazında eklenecekler:

- IFC entity type
- GlobalId
- property sets
- storey/space relations
- quantity takeoff summaries
